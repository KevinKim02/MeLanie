About Us

MeLanie is a machine learning model which uses CNN to determine if an image of a mole is malignant or benign (it will return the likelihood that the mole is malignant). The purpose of the model is to aid people to check moles at home as well as aid doctors in more accurate diagnoses. The model was implemented using Keras, in this environment you are able to load an image, resize it to fit model requirements, scale pixels to a consistent range, and then select and run the pre-trained ML model.
A survey done by The Journal of Clinic and Aesthetic Dermatology found that only 17% of people do a skin check once a year resulting in several cases of skin cancer going undetected until it's too late. By making skin checks easier and more accessible with MeLanie we hope to aid the early detection of melanoma.
